# Casos de Teste BDD - Tela de Login

Este projeto apresenta a escrita de cenários de teste utilizando a abordagem BDD (Behavior-Driven Development) para uma tela de login desenvolvida no Figma.

O objetivo é demonstrar a elaboração de casos de teste funcionais, considerando diferentes comportamentos esperados do sistema a partir dos requisitos visuais da interface.


## Funcionalidades da Tela

A tela de login é composta pelos seguintes elementos:

* Username (Nome de usuário);
* Password (Senha);
* Botão de Login;
* Link "Forgot Password".

## Objetivos do Projeto

* Aplicar conceitos de testes manuais utilizando BDD;
* Elaborar cenários de teste claros e objetivos;
* Validar possíveis fluxos positivos e negativos da funcionalidade de login;
* Demonstrar conhecimentos em análise de requisitos e documentação de testes.

## Cenários de Teste

Os cenários contemplam validações como:

* Login realizado com sucesso;
* Campos obrigatórios não preenchidos;
* Username inválido;
* Password inválida;
* Tentativa de login com dados incorretos;
* Funcionamento do link "Forgot Password";
* Validação do comportamento esperado do botão de Login;
* Validação das mensagens apresentadas ao usuário (quando aplicável).

## Tecnologias e Conceitos Utilizados

* Testes Manuais;
* BDD (Behavior-Driven Development);
* Gherkin;
* Git e GitHub;
* Análise de Requisitos.
